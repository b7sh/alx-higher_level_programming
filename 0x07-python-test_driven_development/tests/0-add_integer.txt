-----------
add integes
-----------

::

	>> add_integer = __import__("0-add_integer").add_integer
	>> add_integer(1, 2)
	3

::
	>> add_integer(2, -1)
	1

also work with floats

::


	>> add_integer(1.0, 2.0)
	3.0

::
	
	>>	add_integer(-2.3, -3.2)
	-5.5

float and non-float

::

	>> add_integer(2.0, 3)
	5

the second argument is optional the default value is 98

::

	>> add_integer(2)
	100
